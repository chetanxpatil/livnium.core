# Semantic Processing

Semantic analysis and meaning extraction from geometric structures.

## Contents

- **`semantic_processor.py`**: Main semantic processing pipeline
- **`feature_extractor.py`**: Extract semantic features
- **`inference_engine.py`**: Semantic inference
- **`meaning_graph.py`**: Graph representation of meaning

## Purpose

This module provides:
- **Semantic features**: Extract meaning from geometric structures
- **Inference**: Derive implicit meanings
- **Meaning graphs**: Represent relationships between concepts
- **Compositional semantics**: Build complex meanings from parts

Used by NLI system and language understanding applications.
