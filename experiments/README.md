# Experiments

Research experiments exploring Livnium's capabilities across different domains.

## Contents

- **`nli/`**: Natural Language Inference
  - Pure geometric reasoning for Entailment/Contradiction/Neutral classification
  - Letter-by-letter encoding with quantum collapse
  - Zero neural networks

- **`quantum_teleportation/`**: Quantum Protocol Tests
  - True quantum teleportation using `TrueQuantumRegister`
  - Bell's inequality violation tests
  - Demonstrates quantum-only capabilities

- **`quantum_core/`**: Quantum-Enhanced Applications
  - AES-128 cryptanalysis using recursive geometry + quantum layer
  - Recursive qubit capacity tests (2.5M+ qubits)
  - Geometric swarm attacks
  - Quantum topology mapping

- **`crypto/`**: Classical Cryptanalysis
  - AES round-by-round analysis
  - Classical search strategies
  - Pattern-based approaches

- **`ramsey/`**: Ramsey Number Solving
  - Geometric basin search for maximum clique-free graphs
  - Dynamic tension fields
  - Checkpoint system for long-running searches

## Purpose

These experiments explore:
- **Language understanding**: Geometric alternatives to transformers
- **Quantum mechanics**: True quantum protocols on classical hardware
- **Cryptanalysis**: Quantum-inspired key search strategies
- **Combinatorial optimization**: Geometric approaches to hard problems

Each experiment includes:
- Implementation code
- Test suites
- Documentation
- Results analysis
